## Project Name
Embedded Web Server

## Project Scope
The scope of this project is to develop a wireless device control system using an ESP32-based web server. It enables remote monitoring and control of electronic devices through a web interface and provides a foundation for IoT and home automation applications.

## Project Description
The Embedded Web Server project uses an ESP32 microcontroller to create a web server that can be accessed through a web browser over a Wi-Fi network. The web page hosted by the ESP32 allows users to monitor and control connected devices remotely. In this project, an LED is controlled using ON and OFF buttons displayed on the web page.

## Components Required

- LED
- 220Ω Resistor

- Jumper Wires
